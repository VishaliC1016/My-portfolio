Projects

Recognizing Eminent Players from Indian Premier League

For this project, I developed a system to identify and rank the top 20 players in the Indian Premier League (IPL) using sentiment analysis of tweets collected from Twitter.

The project involved collecting tweets related to IPL from Twitter's API and performing sentiment analysis using TextBlob for Python. Tweets were categorized into positive, negative, and neutral sentiments using data processing techniques.

Leveraging Convolutional Neural Networks (CNN) implemented with Python's Django framework, I built a model to rank the players based on the sentiment of tweets mentioning them. The CNN model was trained on the sentiment-labeled tweet data to identify the most prominent players based on positive sentiment.

This project not only showcased my skills in natural language processing and machine learning but also demonstrated the practical application of sentiment analysis in determining the popularity and perception of players in a sports context.

Emer-Smart: Emergency Alert System

Emer-Smart is an innovative emergency alert application designed to enhance residential security and safety. Leveraging advanced technology, including the ESP32-CAM microcontroller and doorbell sensor, Emer-Smart provides real-time monitoring and alerts residents in case of suspicious activity or potential threats.

The system operates by continuously monitoring the surroundings using the doorbell sensor. When the sensor detects abnormal activity, such as unauthorized entry or suspicious movement, it triggers the ESP32-CAM to capture images or videos of the event.

Upon capturing the evidence, Emer-Smart sends an immediate alert to the residents via email, providing them with crucial information about the incident. This prompt notification empowers residents to take appropriate actions, such as contacting authorities or alerting neighbors, to mitigate potential risks.

Emer-Smart's integration of cutting-edge technology with proactive security measures makes it an invaluable asset for residential communities, offering peace of mind and enhanced protection against thefts and intrusions.
